Instructions for creating a new record

(2) Apply your annotation to this text.
On the edit page:
- Select the option Source above the "long-format prose narrative."
This will present you with the narrative as "source code."

- Hold down COMMAND + F. This will bring up a Find bar.
Input the text for which you have created an annotation, and hit Enter.
Your text will appear highlighted in the narrative.

- Substitute the following formula in for the highlighted text:
<span>data-neatline-slug=YOUR SLUG GOES HERE>THE TEXT GOES HERE</span>.
